Web Dizayn

Sayt dizaynın saylaw

Dizayn nima?

Web dizaynda sayttıń sırtqı kórinisin hám qolaylılıǵın jaratıw áhmiyetli bolıp esaplanadı.

Reń, shrift hám jaylastırıw

  • Reńler - ingliz tilindegi awdarmaları yamasa kod arqalı beriledi
  • Shriftler tema hám tekstke qarap ózgeredi hám saylanadı
  • Video, súwret hám tekstlerdiń durıs jaylasıwı sayttıń sapasın arttıradı

Fon reńi

Fon reńin saylawda sayt temasına hám tekstlerge áhmiyet beriw kerek

Tegler

  • b tegi tekstlerdi qalıń jazıwǵa aylandırıp beredi
  • u tegi tekstlerdiń astın sızıp beredı
  • i tegi tekstlerdi qıya qılıp beredi

Atributlar

  • Hár bir teg style atributı arqalı óz stiline iye boladı
  • Tekstlerge font sózi menen dizayn beriledi(font-family, font-size, )
  • Fonǵa background-color atributı járdeminde dizayn beriledi

Trenddegi shriftlar

  • Roboto
  • Open Sans
  • Inter
  • Montserrat
  • Poppins

Trenddegi reńler

  • Neytral reńler- Krem reń, kúl reń
  • Tábiyat reńleri - Jasıl, kók
  • Qarańǵı reńler - Dark gray
  • Gradientler - Purple

Jaylastırıw

Jaylastırıw atributları: left,right, center. Tekst jaylastırıw ushın text-align atributı qollanıladı.
